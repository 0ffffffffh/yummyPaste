you can use 

c style byte array, 
***{0x90, 0x90, 0x90};***

c style shellcode,
***"\x90\x90\x90"***

decimal array
***144, 144, 144*** (can be surrounded with curlies).

or sequence of a decimal numbers.

***144 144 144***
